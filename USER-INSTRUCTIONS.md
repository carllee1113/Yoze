# YOZE 使用说明（批量设置版本）

## 1. 你可以一次设置多种药

在首页点击 `+` 或相机按钮后，会进入 **批量设置药物** 页面。

你可以在同一轮里混合添加：
- 手动新增（点击 `手动新增`）
- 拍照识别（点击 `拍照新增`）

## 2. 填写每种药的资料

每种药都可编辑以下字段：
- 药物名称
- 形式（药丸 / 胶囊 / 药水 / 膏药）
- 剂量
- 服用形式（饭前 / 饭后 / 空肚食 / 饱肚食）
- 每次份量（1~4）
- 每日次数（1~6）
- 颜色标签

## 3. 统一设置每日起始时间

在页面上方点击 `每日开始时间`，选择这批药的一天起始时间（例如 08:00）。

系统会按最高频次自动生成共享时段，默认每 4 小时一个时段。

例：
- A 药 3 次/日，B 药 4 次/日
- 起始时间 08:00
- 时段：08:00、12:00、16:00、20:00

## 4. 查看“服药分配预览”

在保存前可先看每个时段包含哪些药。

确认无误后点击 `保存排程`。

## 5. 保存后会发生什么

- 系统会把每个时段的药物写入排程
- 自动建立提醒通知
- 返回首页显示当天服药进度

## 6. 首页提醒卡片（大卡 + 小卡）

- 每个服药时段是一张大卡（显示时间）
- 大卡内每种药是一张小卡
- 大卡右侧有 `全部` 勾选框
- 勾选大卡：该时段所有小卡自动变为完成
- 全部小卡都完成：大卡自动变为完成
- 未完成时，小卡会显示 `等待中`

## 7. 今天服药进度计算

- 分母：今天总时段数（按最高频次决定）
- 例：A 每日 2 次，B 每日 7 次，分母 = 7
- 分子：完成的大卡数量
- 只有大卡完成，分子才会加 1

## 8. 月历历史标记规则

- 100%：绿色勾
- >= 80% 且 < 100%：蓝色勾
- < 80%：红色圆点

## 9. 常见操作

- 删除某种药：点击药物卡片右上角垃圾桶
- 重新拍照补充药物：点击 `拍照新增`
- 只手动设置：只用 `手动新增` 即可

## 10. 建议

- 拍照时尽量一张标签对应一种药，识别更稳定
- 保存前先检查每日次数与每次份量是否正确
